Forward

Welcome to my series of python articles for absolute beginers. The articles below are designed to help people with no experience (other than using a computer) to get started with programming, and particularly with python. They are written with the aim of helping people coming from a social science background to transition away from programs like SPSS and MS Excel and so a lot of the examples are childishly simple, especially in the earlier articles.

If you are new to python, I'm sure you will enjoy the journey. Just be patient and remember that its OK if you need a few tutorials, from a few different sources, before something clicks. Sometimes you have hear the same thing said multiple different ways before it perculates down into your head. If you find that my explanaitions don't work for you I can really hightly recomend these youtube tutorials and this book. They really helped me advance when I was just starting out.

If you're a seasoned python data scientist, then please look on this site with a kind eye, my aim has been to break everything down so that nothing is taken for granted or assumed because I found that people who are good at coding tend to take a lot for granted when teaching new people. This can be really disheartening and leave new folk feeling like they will never progress without a comp-sci degree and a background in building linux systems.

********* Content **********

  1. The Mindshift. This is a quick post about a mindset that might be useful to you as you begin to journey into working in code.

  2. Setting up a place to work. Get python installed and take a quick tour of Spyder, the main tool you'll be using as you follow these posts.

  3. Starting with strings. We begin working with actual information in python by looking at how python handles everyday English. This is where you are introduced to your first python data class, the string.

  4. Numbers in python. From normal everyday English language, we move onto normal everyday numbers and you are introduced to two new python data classes; ints and floats.

  5. Python Variables. Now that we have a few data classes under our belt, we can start to think about assigning information, like strings or numbers, to python variables.

Check back soon for more beginers python posts.

Hold on tight, you got this!

Kev
